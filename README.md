## 📌 Что это?

Нейрободр — это десктопное приложение с графическим интерфейсом для регистрации и авторизации операторов с биометрической привязкой. Программа позволяет:

- ✅ Создавать учетные записи операторов
- ✅ Загружать и верифицировать фотографии
- ✅ Проверять соответствие размера изображений (800x600 px)
- ✅ Авторизовывать операторов по ID
- ✅ Отслеживать время работы и состояние

## ⚙️ Как работает?

### 🖥️ Основные окна:

#### 🏁 Окно №1 - Стартовая форма
Выбор режима работы: регистрация или авторизация.

#### 📝 Форма регистрации
• Ввод ФИО и возраста

• Сохранение в CSV с автоматическим присвоением ID

• Загрузка фото оператора

#### 🔐 Авторизация
• Ввод ID оператора

• Отображение информации об операторе

#### ✅ Верификация
• Проверка фото на соответствие размеру

• Отображение результата проверки


### 📁 Структура данных:
• operation_db.csv - база данных операторов

• operations/ - папка с фотографиями операторов (ID_номер.jpg)
